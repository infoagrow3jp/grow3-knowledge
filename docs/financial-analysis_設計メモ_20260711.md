# financial-analysis 設計メモ

## 1. 目的

財務分析サブエージェント（名称：`financial-analysis`）を新設する。役割は投資・融資の助言ではなく、経営判断材料の作成である。

## 2. 構成

スキル（指標定義書＋判断基準）＋サブエージェント＋evalの三点セットで構成する。

**五層構造：**

1. 時代前提
2. 実態修正（資産実在性スクリーニングを含む）
3. 守りの診断
4. 攻めのMQミックス（商品軸＋顧客軸・制約資源あたりMQ）
5. クロス判定（社労士・SSR・評価制度接続）

**横串（全層に適用）：**

- 責任境界：税理士法・融資可否の非断定
- 守秘：実データは公開リポジトリに置かない

**時代前提の扱い：**

分析基準日時点の経営環境を確認する。2026年はコスト上昇を踏まえ、P戦略を優先的な検討候補とするが、企業ごとの競争環境・顧客構成・制約条件を確認して判断する。環境前提と内部目安は年1回以上見直す。

## 3. 指標定義の決定

- 簡易法はFCFと呼ばず「簡易営業CF」とする
- 実績営業CF・FCF・返済原資CFは分離し、それぞれ計算値・調整項目・調整理由の3列で構成する。符号規約を固定する
- 債務償還年数は分子・分母の型を明示し、算式を併記する
- 公的定義の労働分配率と、MQ人件費比率（人件費÷M）は別指標として区別する

## 4. 運用設計

**3分析モード：**

1. 財務会計
2. MQ管理会計
3. 人件費・賃上げ余力

**工程3段：**

1. 入力検証
2. 計算（コード実行。暗算禁止）
3. 見立て（事実→解釈候補→追加確認→打ち手候補）

**閾値の根拠区分（5種）：**

- official
- statistical
- financial_institution
- grow3_heuristic
- client_target

## 5. 資産実在性スクリーニング

**観察対象：**

- 繰越損失×流動資産膨張
- 回転期間3期推移
- 黒字×営業CF連続マイナス
- 雑勘定・役員貸付
- 投資有価証券ウエイト

**原則：**

各シグナルは不正・粉飾の判定ではなく、実在性・回収可能性・時価・滞留状況を追加確認する契機として扱う。単一指標で判定せず、3期推移と補助資料を組み合わせる。出力は断定せず確認質問に変換する。

**確認資料の例：**

- 売掛金年齢表・回収予定
- 棚卸資産明細・滞留期間
- 預金残高資料
- 固定資産台帳・減価償却明細
- 投資有価証券の時価・保有目的
- 仮払金・貸付金などの相手先と発生理由
