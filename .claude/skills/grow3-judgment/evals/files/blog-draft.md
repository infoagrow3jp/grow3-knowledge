# 【テスト素材・架空記事】任せたのに戻ってくる仕事

※eval用に欠陥を意図的に仕込んだ架空の下書きです。公開・転用不可。

「課長に任せたはずの案件が、気づけば自分の机に戻ってきている」

ある製造業の社長から、こんな話を聞きました。
権限は渡した。役割も伝えた。それなのに、
最後の判断だけがいつも社長のところに戻ってくる。

課長のやる気の問題でしょうか。私はそうは思いません。
多くの場合、任せた仕事の「判断基準」が言葉になっていないのです。
どこまで自分で決めてよいのか、何を相談すべきなのか。
その線引きがないまま任せると、部下は安全側に倒れて確認に来ます。

ところで、評価制度の話もしておきましょう。評価基準が曖昧な会社では
賞与の時期に不満が出やすく、等級制度の整備が急務です。評価者研修も
セットで行うことが望ましく、私たちは経営者の想いに寄り添う支援を
心がけています。

任せるとは、仕事を渡すことではなく、判断の基準を渡すことです。
