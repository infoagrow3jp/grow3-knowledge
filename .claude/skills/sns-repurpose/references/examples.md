# 判断事例集：SNS再設計

## 事例：複数投稿の「断面」判定

- 状況の型：1つの素材（ブログ記事等）から複数本の投稿を生成するとき。
- 誤りやすい出力：同じ冒頭・同じ構成の要約分割（実質同文の複製）。
  cta_levelの指定と本文の温度が一致していない（theme指定なのに接続文がない）。
- 正しい出力：各投稿は素材の異なる断面（別の場面・別の違和感）から起こす。
  生成後に全投稿の冒頭3行を並べて重複を自己チェックし、cta_levelと本文の
  接続温度を照合する。
- 判断の分かれ目：N本は「要約のN分割」か「N個の断面」か。冒頭同士を
  入れ替えても成立するなら、断面になっていない。
