# 【テスト素材・架空記事】会議で決まったことが、なぜ動かないのか

※eval用の架空記事です。記事内の社名・事例はすべて架空です。

「会議では全員がうなずいていたのに、翌週になっても誰も動いていない」

株式会社テスト製作所（従業員45名・金属加工業）の山田社長から、
こんな相談を受けたことがあります。月曜の経営会議で決めたはずの
改善活動が、金曜になっても始まっていない。誰かが反対したわけでも
ない。それなのに、動かない。

社長は「うちの管理職は当事者意識が足りない」と言いました。
私は少し違う見方をしています。

会議で決まったのは「何をやるか」だけで、「誰が・いつまでに・
どこまでやるか」が決まっていなかったのです。全員が賛成した案件ほど、
全員が「誰かがやるだろう」と思っている。責任の所在が言葉になって
いない案件は、善意の人たちの間で静かに止まります。

テスト製作所では、会議の最後に「決定事項・担当・期限・次に報告する場」
を1枚で確認する型を入れました。それだけで、翌月から改善活動が
動き始めました。管理職の意識が変わったのではありません。
動ける形に、決定が翻訳されただけです。

決まったのに動かないとき、足りないのは当事者意識ではなく、
決定を動きに翻訳する一手間かもしれません。
