# 第1部　設計の思想

**Grow3の設計哲学（前提に置く価値観）**
- 体験 → 名前 → 実験：まず体験させ、概念で「名前」をつけ、現場で試させる。講義で答えを渡さない。
- 講義より対話：ワーク比率の目安は約半分以上（〜55%）。座学は最小限。
- 設計の主語はレベル3＝行動：満足度（レベル1）でなく、現場で行動が変わるかで設計する。[→F:研修効果・学習転移]
- ピーク・エンド：山場と締め方を意図的に置く。平均でなく頂点と終わりが記憶を決める。[→F:行動経済学]
- 人は正しさだけでは動かない：欲求・感情に触れて初めて動く。[→F:欲求プロフィール][→F:選択理論心理学]
- 振り返りが生命線：体験を学びに変えるのは内省と概念化。[→F:経験学習サイクル][→F:メタ認知]

# 第2部　設計プロセス

**ADDIE（基本の5段階）**
- 分析（Analysis）：誰に・何の課題・どんな行動変容を狙うか。発注者と受講者のニーズのズレを掴む。[→F:認知の非対称性][→F:デザイン思考]
- 設計（Design）：ゴール（行動）→構造→ワーク→時間 の順で決める。
- 開発（Development）：PPTX・ワークシート等の成果物を作る。
- 実施（Implementation）：当日ファシリ。
- 評価（Evaluation）：レベル1〜4で効果を見る。[→F:研修効果・学習転移]

**SAM（反復型・補完）**
完璧な計画より、小さく作って試して直す。試作→検証を回す。[→F:デザイン思考][→F:経験学習サイクル]

**Grow3版 設計7ステップ（実務の流れ）**
1. ゴール定義：終了時に受講者がどう変わるか（＝観察できる行動で書く）
2. 見立て：受講者・発注者を読む（思考特性／発達段階／現場の制約）[→F:思考特性4象限][→F:成人発達理論]
3. 1メッセージ決定：この研修で一番持ち帰る一言に絞る
4. 構造設計：オープニング→本編ブロック→クロージング
5. ワーク設計：体験→名前→実験を配置
6. 時間設計：タイムスケジュール（TS）を組む
7. 成果物化：PPTX／TS／ファシリガイド／ワークシート／振り返り

# 第3部　構造の型

**時間設計パターン**
- 標準尺：半日（約3.5h）／1日（6〜7h）。
- 大枠：オープニング（つかみ・安全・目的）→本編（複数ブロック）→クロージング（振り返り・行動宣言）。
- 本編1ブロックの5拍：導入 → ワーク → 共有 → 概念化（名前づけ）→ 転用（現場でどう使うか）。[→F:経験学習サイクル][→F:SECIモデル]
- TSは、指定尺（休憩含む）・各パート合計・各パート内訳の3段で検算し、すべて一致するまで確定しない。
- 時間を増減した場合は前案との差分を確認し、増減合計が全体差分と一致することを確認する。

**ワークの型**
- 単位：個人 → ペア → グループ → 全体共有（広げて→絞る）。[→F:デザイン思考（ダブルダイヤモンド）]
- 代表ワーク：グループケーススタディ（例「○○主任の一日」）、n択の行動宣言、偏愛マップ、付箋の発散→分類、ロールプレイ、自己診断。
- 原則：発散と収束を混ぜない。問いで進める。[→F:アクションラーニング]

**場の設計**
- 冒頭で心理的安全性を作る（自己開示・グランドルール・問い）。[→F:心理的安全性][→F:タックマン]
- 対立・混乱は避けず、安全に通す（混乱期は通過点）。[→F:タックマン][→F:アサーティブネス]

# 第5部　ファシリテーションの型

**問いの順序（ORID）**
- O 客観：何が起きたか（事実）
- R 反応：どう感じたか（感情）
- I 解釈：それはどういう意味か（意味づけ）
- D 決定：次に何をするか（行動）
- → 経験学習の内省〜概念化〜実験と対応。[→F:経験学習サイクル]

**進め方の原則**
- 助言でなく問いで進める。[→F:アクションラーニング][→F:社会構成主義]
- 発散と収束のリズムを意識的に切り替える。[→F:デザイン思考]
- 沈黙は待つ、脱線は拾い直す、対立は安全に扱う。[→F:タックマン][→F:アサーティブネス]

# 第6部　成果物と品質基準

**標準成果物セット**
- PPTX（スライドマスター準拠）
- タイムスケジュール（TS／Word・Excel、計算式は保持）
- ファシリテーターガイド
- ワークシート
- 振り返りシート（＋必要に応じ研修概要）

**品質基準（採点）**
詳細は `grow3-training-scoring-axes.md`（研修採点軸）を参照。設計思想は2段階。
- 汎用軸（素材タイプ別）：スライドデッキ／ケースワークシート／振り返りシートごとに5軸。
- テーマ固有軸（研修ごと）：核心テーマ・骨格に沿って5軸を別途設定（例：ナイカイ「任せると責任」＝モノサシ概念の一貫性／タイムライン整合／伝わりやすさ／ワーク接続／実施可能性）。
- 原則：採点前に使う軸を宣言／各軸20点×5＝100点／改善FBは代替案とセット／情報不足の軸は「未確認」で保留。
